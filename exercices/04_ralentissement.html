Commentez.

On commence par dterminer la vitesse du centre de masse B : $$ \vec{v_B} = \frac{m \vec{v_{in}}+M \vec{V_{in}}}{m+M} = \frac{1}{1+A} \vec{v_{in}} $$

car $ \vec{V_{in}}= \vec{0} $

On peut exprimer les vitesses du neutrons $\vec{w_{in}} $ et du noyau $\vec{W_{in}}$ dans le centre de masse $$\vec{w_{in}} = \vec{v_{in}} - \vec{v_{B}} = \vec{v_{in}} (1 - \frac{1}{1+A}) = \frac {A}{A+1} \vec{v_{in}} $$ $$\vec{W_{in}} = \vec{V_{in}} - \vec{v_{B}} = \vec{v_{in}} (\frac{A}{1+A} - 1 ) = - \frac {1}{A+1} \vec{v_{in}}$$

Dans le rfrentiel du laboratoire, l'nergie cintique avant le choc vaut :

$$ E_c = E_{c1} + E_{c2} = \frac{1}{2} m v_{in}^2 $$

car $ E_{c2}=0 $ car $ \vec{V_{in}}= \vec{0} $

Dans le rfrentiel du centre de masse, l'nergie cintique vaut:

$$ E_B = E_{c1}' + E_{c2}' = \frac{1}{2} m (\frac{A}{1+A})^2 v_{in}^2 + \frac{1}{2} M (\frac{1}{1+A})^2 v_{in}^2 = \frac {A}{A+1} E_c$$

On exprime la quantit de mouvement : $$ \vec{p_t} = m \vec{w_{in}} + M \vec{W_{in}} = \vec{0}$$

Conservation quantit de mouvement :

$$\frac{d\vec{p}}{dt} = \vec{0} \Rightarrow \vec{p} = \vec{cste} \Rightarrow m \vec{w_{out}} + M \vec{W_{out}} = \vec{0} = m \vec{w_{in}} + M \vec{W_{in}}$$

Conservation de l'nergie cintique :

$$\frac{1}{2} m w_{in}^2 + \frac{1}{2} M W_{in}^2 = \frac{1}{2} m w_{out}^2 + \frac{1}{2} M W_{out}^2$$

Ce qui permet d'obtenir aprs manipulation :

$$ \begin{cases} w_{out}^2 = w_{in}^2 \\ W_{out}^2 = W_{in}^2 \end{cases} $$

On considre $\theta$ connu et l'on cherche $\vec{v_{out}}$. Pour cela il faut une amplitude et une direction
D'aprs la question 2 :

$$\vec{v_{out}} = \vec{w_{out}} + \vec{v_{B}} \Rightarrow v_{out}^2 = w_{out}^2 + v_{B}^2 + 2 \vec{w_{out}} \cdot \vec{v_{B}}$$

$$v_{out}^2 = v_{in}^2 [(\frac{A}{A+1})^2 + \frac{2A}{(A+1)^2} \cos(\theta) + \frac{1}{(1+A)^2}] = v_{in}^2[\frac{A^2 + 2A \cos(\theta) + 1}{(A+1)^2}]$$

On calcule aussi le fraction d'erngie cintique en sortie par rapport celle en entre :

$$ \frac{E_{c_{out}}}{E_{c_{in}}} = \frac{1 + 2A \cos(\theta) + A^2}{(A+1)^2}$$

$$\frac{E_{out}}{E_{in}}=\frac{A^2+2A\mathrm{cos}(\theta)+1}{(A+1)^2}$$

import numpy as np

def f(theta,Ein=1,A=1):
    """Calcul de l'nergie post-choc en fonction de l'angle de choc et du rapport de masse A

    Args:
        theta (int): _angle de choc_
        Ein (int, optional): Energie pr-choc. Defaults to 1.
        A (int, optional): Rapport de masse. Defaults to 1.

    Returns:
        int: Energie post-choc
    """
    Eout=Ein*(A**2+2*np.cos(theta)*A+1)/(1+A)**2
    return Eout

import multiprocessing
from multiprocessing import Pool
import numpy as np

def evolution_neutron(args):
    """Simule la vie d'un neutron

    Args:
        args (tuple): E_source,E_lower_limit,alpha,intervalles
        E_source (float): Energie initiale du neutron
        E_lower_limit (float): Energie minimale du neutron
        alpha (float): Coefficient de ralentissement
        intervalles (np.array): Intervalles d'nergie

    Returns:
        np.array: Score local des intervalles d'nergie
    """
    E_source, E_lower_limit, alpha, intervalles = args
    energie=E_source
    score_local = np.zeros(len(intervalles))
    score_local[-1] = 1

    while energie>E_lower_limit:
        index_intervalle = np.searchsorted(intervalles, energie) - 1
        if 0 <= index_intervalle < len(score_local):
            score_local[index_intervalle] += 1
        energie=rng.uniform(alpha*energie,energie)
    return(score_local)

import multiprocessing
from multiprocessing import Pool
import numpy as np
import matplotlib.pyplot as plt
processes=multiprocessing.cpu_count()


def evolution_neutron(args):
    """Simule la vie d'un neutron

    Args:
        args (tuple): E_source,E_lower_limit,alpha,intervalles
        E_source (float): Energie initiale du neutron
        E_lower_limit (float): Energie minimale du neutron
        alpha (float): Coefficient de ralentissement
        intervalles (np.array): Intervalles d'nergie
        Proba_absorption (float): Probabilit d'absorption par collision

    Returns:
        np.array: Score local des intervalles d'nergie
    """
    E_source, E_lower_limit, alpha, intervalles, Proba_absorption = args
    energie=E_source
    score_local = np.zeros(len(intervalles))

    score_local[-1] += 1  # Compter le neutron initial dans le dernier intervalle

    while energie>E_lower_limit:
        if rng.uniform(0,1) < Proba_absorption: # On absorbe le neutron avec la probabilit donne
            break # Si il est absorb, on arrte la simulation de ce neutron
        energie=rng.uniform(alpha*energie,energie)
        index_intervalle = np.searchsorted(intervalles, energie) - 1
        if 0 <= index_intervalle < len(score_local):
            score_local[index_intervalle] += 1
    return(score_local)
